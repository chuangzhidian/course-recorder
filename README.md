# course-recorder · 在线课程挂机录屏工具

自动化"点下一节 + 播放"，配合 OBS 把整门加密课程（画面 + 声音）录成本地视频。

> **用途边界**：本工具**不下载、不解密**任何视频，只是自动化你手动就能做的操作（切课时、点播放）。
> 仅用于录制**你已购买 / 有权观看**的内容，供个人离线学习。加密视频（如保利威 .pts）无法直接下载，
> 录屏是唯一合规可行的离线方式。

---

## 适用平台

| 脚本 | 平台 | 说明 |
|------|------|------|
| `shenlan_autoplay.user.js` | 深蓝学院 shenlanxueyuan.com | 视频为保利威(Polyv)加密流(.pts)，无法下载，用录屏 |

深蓝的播放器是**同源 iframe**（`#task-content-iframe`），视频 `<video>` 可读，所以能用 `ended` 事件精准连播。

## 一次性准备

### 1. 装油猴脚本
- 浏览器装 [Tampermonkey](https://www.tampermonkey.net/) 扩展；
- Tampermonkey → 新建脚本 → 把 `shenlan_autoplay.user.js` 内容整段粘进去保存。

### 2. 装 BlackHole（内录系统声音，免费）
```bash
brew install blackhole-2ch
```
装完在"音频MIDI设置"里能看到 **BlackHole 2ch** 设备。

### 3. 装 OBS（免费）
```bash
brew install --cask obs
```

## 每次录制流程

### 第 1 步：让浏览器允许"无手势有声自动播放"
无人值守时脚本要自动 `play()` 带声音的视频，浏览器默认会拦。先 `Cmd+Q` **完全退出 Chrome**，再用 flag 启动（沿用你原来的登录态）：
```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --autoplay-policy=no-user-gesture-required
```
然后正常进课程页。

### 第 2 步：设置声音（决定吵不吵）
- **挂机睡觉（静音录制）**：`系统设置 → 声音 → 输出` 选 **BlackHole 2ch**。
  房间里没声音，但 OBS 能从 BlackHole 录到音轨。
- **想边录边听**：在"音频MIDI设置"里建一个"多输出设备"= BlackHole + 你的扬声器/耳机，输出选它。

### 第 3 步：配 OBS
- 来源：**窗口捕获** → 选浏览器窗口（切别的窗口不影响录制）；
- 音频：设置 → 音频 → 桌面音频 → 选 **BlackHole 2ch**；
- 输出：设置 → 输出 → 录像 → 打开 **自动分段**（按大小或时长切文件，避免单个文件过大）；
- 分辨率按视频原生大小，帧率 30 即可（讲课/PPT 类足够）。

### 第 4 步：开录 + 启动连播
1. OBS 点 **开始录制**；
2. 回到课程页，点右下角面板 **▶ 从第1节**；
3. 脚本自动：开播 → 播完等 4 秒 → 切下一节 → 再开播 …… 直到 40 节全部放完。

### 第 5 步：录完后切成单节文件
点面板 **📋 分段表** 复制"课时→开始时间"清单，按这些时间点在剪辑软件（如 剪映 / ffmpeg）里把长录像切成一节一节。

## 控制面板（右下角）

| 按钮 | 作用 |
|------|------|
| ▶ 从第1节 | 清空分段表，从第 1 个视频课时开始自动连播 |
| ▶ 从当前节 | 从你当前打开的这节开始 |
| ⏹ 停止 | 停止自动连播 |
| ⏭ 手动下一节 | 手动跳到下一节 |
| 📋 分段表 | 复制每节开始时间，供录后切片 |

状态、当前节、最近日志都显示在面板里；控制台里 `[SL-AUTO]` 开头是详细日志。

---

## 可调参数（脚本顶部）

| 参数 | 默认 | 说明 |
|------|------|------|
| `BUFFER_MS` | 4000 | 一节结束后等几秒再切下一节（给录屏留干净分段边界） |
| `SAFETY_PAD_MS` | 120s | 单节看守 = 视频时长 + 该余量，防 `ended` 不触发时卡住 |
| `SAFETY_FALLBACK_MS` | 75min | 视频时长未知时的单节最长看守 |

## 排障

- **不自动开播**：确认第 1 步的 Chrome flag 生效了；或播放器需要点海报——脚本已尝试点常见播放按钮，若仍不行把控制台 `[SL-AUTO]` 日志发出来对选择器。
- **切换后跑偏 / 重复**：脚本用 `/task/<id>/` 锚定当前节，一般不会跑偏；若整页刷新丢状态，它会用 localStorage 续上。
- **录像没声音**：检查系统输出是不是 BlackHole、OBS 桌面音频是不是也选了 BlackHole。
